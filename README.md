# robotis-convivium-demo

Static hosting for the **Phase 1 end-to-end demo** of the Robot Video → LeRobot
pipeline. The page (`index.html`) is a self-contained, dependency-free walkthrough
generated from a real `examples/run_phase1.py` run — pipeline overview, live run
output, the byte-for-byte input/output artifact, the trainability ladder, and the
three-phase roadmap. Section titles are bilingual (English / Korean).

**Live page:** https://e3f-six.github.io/robotis-convivium-demo/

The pipeline source, design docs, and tests live in the main (private) repository.
